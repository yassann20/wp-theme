スキル

  • HTML

    HTMLは幅広いタグを扱うことができ、それぞれの目的に合ったタグを使用することでSEO対策にも貢献させることができます。

  • CSS

    最近では素のCSSだけではなく、より汎用性に富んだSASS(SCSS形式)を使用しコーディングまで対応することができます。

  • javascript

    javascript及びjqueryについては主に動きを加える場合に使用しており、背景アニメーションのような高度な幾何学模様を制作することも可能です。

  • php

    お問い合わせページのフルスクラッチ開発、wordpress内でのループの作成やカスタマイザーの開発等幅広く対応可能です。

  • wordpress

    wordpressでは既存のテンプレートを使用したサイト構築から、テーマカスタマイザーを使用した自作テンプレート制作まで幅広く対応可能です。

  • mysql

    mysqlではphpと連携し、商品管理サイトの作成やwordpress回りのを構築する際に使用しています。

  • git

    ソースコードの管理はgitを使用していますのでチームでの開発を行う案件にも対応可能です。

  • photoshop

    デザインデータの書き出し等を主に使用しています。また撮影した画像などの加工処理等も行えます。

  • illustrator

    主にイラストやロゴの制作に使用しており、基本的な操作をすることが可能です。

  • XD

    主にデザインを書き出す際に使用するツールでphotoshopの次くらいの頻度で書き出すことが多いです。またこれ以外のデザインカンプの共有法としてはfigmaからの
